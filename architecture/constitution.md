# NOAH Constitution

1. Runtime before Features.
2. Architecture before Code.
3. Contracts before Implementations.
4. Composition before Coupling.
5. Offline before Cloud.
6. AI is Optional.
7. Security before Convenience.
8. One Responsibility per Component.
9. Every Version is Complete.
10. Long-Term Stability over Short-Term Speed.

## Core Rule

The Runtime must never depend on AI.

AI is always optional.
AI is always a plugin.
AI is never the core.
